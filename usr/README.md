# Chirok

Chirok is an interactive storytelling platform that combines social media-style scrolling with playable, branching narrative experiences. It acts as a "story engine," enabling creators to build interactive short films and viewers to seamlessly transition from preview trailers into playable stories.

## Features

- **Preview to Play**: A feed of vertical trailers where users can immediately start an interactive story with zero loading screens.
- **Hardware Integration**: The interactive viewer leverages device sensors (like the accelerometer) for immersive storytelling and dynamic choices.
- **Creator Studio**: A built-in node graph editor for storytellers to visually assemble clips and logic blocks without writing code.

## Tech Stack

- **Framework**: Flutter (Dart)
- **State Management**: Provider
- **Sensors**: `sensors_plus` plugin for tilt controls

## Getting Started

1. Clone the repository.
2. Run `flutter pub get`.
3. Launch with `flutter run` on a mobile device or emulator for the best sensor experience.
